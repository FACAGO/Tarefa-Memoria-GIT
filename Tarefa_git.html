Apuntes GIT

Con GIT faise moito máis fácil e dinámico traballar dende un repositorio local a un repositorio remoto

Por exemplo:

GIT COMMIT:
Crea unha nova rama
GIT BRANCH
Na mesma rama crea outra referencia co nome que queiramos
GIT CHECKOUT
Cambia da rama na que tamos a nova referencia creada
GIT MERGE
Une os dous puntos
